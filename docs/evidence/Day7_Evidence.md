## 5. 遇到的问题与解决方法

**JSON Server 端口切换**：之前使用的是 3000 端口，后改为 3001，需要同步更新 http.ts 中的 baseURL。

**git revert 误操作**：Day3 提交后用户要求撤回但保留代码改动，使用了 revert + revert 反向操作 + soft reset 的方式，导致 git 历史较为复杂，但代码状态正确。
